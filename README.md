# Cellanome Investment Research Platform

An interactive investment research report platform showcasing biotech company analysis with AI-powered Q&A capabilities.

## Overview

Cellanome is a comprehensive research platform featuring:
- **Interactive data visualizations** using Recharts
- **Editorial-style design** inspired by Nature Portfolio aesthetics  
- **AI-powered assistant** for answering questions about the research
- **Responsive layout** optimized for investor presentations

## Tech Stack

- **Frontend:** React 19 + TypeScript + Vite
- **Styling:** Tailwind CSS + Radix UI components
- **Charts:** Recharts for data visualization
- **Backend:** Express.js (minimal API layer)
- **Design:** "Nature Portfolio" editorial theme with teal/amber palette

## Getting Started

### Prerequisites
- Node.js 18+
- pnpm (recommended)

### Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Features

- **Funding Analysis:** Interactive timeline and rounds breakdown
- **Market Sizing:** Multi-year projections with trend analysis
- **Competitive Landscape:** Radar charts and positioning matrices
- **Patent Portfolio:** Timeline visualization of IP development
- **AI Assistant:** Coming soon - interactive Q&A about research data

## Design Philosophy

The platform uses the "Editorial Life Sciences" design approach:
- Clean white canvas with strategic teal (#0d5c63) and amber (#e8a020) accents
- Typography-led hierarchy (Playfair Display + DM Sans)
- Generous whitespace with asymmetric editorial grid
- Data visualizations as editorial graphics
